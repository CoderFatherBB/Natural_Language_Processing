## About

This repository currently centers on a bank complaints classification project. It includes the raw complaint data, cleaned dataset, vectorizer artifacts, trained model artifact, and the training notebook.

## Repository Map

- `NLP - Bank Complaints Detection/Training.ipynb`
- `NLP - Bank Complaints Detection/complaints-2021-05-14_08_16.json`
- `NLP - Bank Complaints Detection/cleaned_complaints.csv`
- `NLP - Bank Complaints Detection/count_vector.pkl`
- `NLP - Bank Complaints Detection/tfidf.pkl`
- `NLP - Bank Complaints Detection/nb_model.pkl`


## Getting Started

Clone the repository:

```bash
git clone https://github.com/CoderFatherBB/Natural_Language_Processing.git
cd Natural_Language_Processing
```

Open the folder or notebook you want to explore. Most directories are self-contained, so start from the local `README.md`, source file, notebook, or project folder for that topic.
